# Plugin Market Specification

This section lists the AstrBot plugin-market JSON protocol accepted by the current Dashboard and backend.

> [!IMPORTANT]
> This modernized fork does not currently operate an independent official plugin
> marketplace under Xero-Team. The default registry still points to upstream AstrBot
> services for compatibility with existing browse, install, and update flows. Upstream
> names and links in these specifications do not mean that this fork operates or
> endorses those services. Custom registries are supported, but this section does not
> designate any custom source as this fork's official marketplace.
