# 插件市场规范

本节列出当前 Dashboard 和后端所接受的 AstrBot 插件市场 JSON 协议规范。

> [!IMPORTANT]
> 本现代化 fork 当前没有由 Xero-Team 运营的独立官方插件市场。默认市场源仍指向
> AstrBot 上游服务，以兼容现有的浏览、安装和更新流程；规范中的上游名称与链接不
> 表示这些服务由本 fork 运营或背书。系统也支持自定义 registry，但本节不会将任何
> 自定义源指定为本 fork 的官方市场。
